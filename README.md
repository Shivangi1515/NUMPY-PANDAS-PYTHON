# NumPy & Pandas Learning Repository

This repository is created to practice and understand core concepts of **NumPy** and **Pandas** used in data analysis.

## NumPy Concepts
- Creating arrays
- Array indexing and slicing
- Mathematical operations
- Shape and reshape

## Pandas Concepts
- Working with Series and DataFrames
- Handling missing values
- Data filtering and sorting
- Reading CSV files

## Goal
To build a strong foundation in Python data analysis libraries.
